# 🥟 Pastel & Cana 

O **Pastel & Cana** é uma interface de autoatendimento (PWA) desenvolvida para o ecossistema **learnTECH**. O projeto foca em uma experiência de usuário (UX) fluida, inspirada em totens de quiosques modernos, utilizando uma estética Cyber-Retro com elementos de glassmorphism.

## 🚀 Tecnologias e Metodologias

- **Frontend:** HTML5, CSS3 (Variáveis e Flexbox/Grid) e JavaScript Vanilla.
- **Design:** Estética Cyber-Retro com neon green, pixel art e sombras suaves (drop-shadow).
- **Metodologia:** Aplicado o princípio de **Lean** e **80/20 (Pareto)** para entrega de valor estratégico com código eficiente.
- **Arquitetura:** Separação clara entre lógica de dados (`database.js`) e lógica de renderização (`script.js`).

## 📋 Funcionalidades

- **Carrossel de Destaques:** Banner interativo com produtos premium e call-to-action direto.
- **Filtros Dinâmicos:** Navegação por categorias (Tradicionais, Especiais, Bebidas) e filtro de Lançamentos.
- **Cardápio Escalável:** Estrutura preparada para suportar mais de 40 itens com carregamento otimizado.
- **Foco Natural:** Seleção de bebidas focada em insumos naturais como Caldo de Cana e Água de Coco.

## 📂 Estrutura de Arquivos

```text
├── assets/             # Imagens dos produtos (Pastéis, Caldos e Combos)
├── index.html          # Estrutura principal do quiosque
├── script.js           # Inteligência e manipulação do DOM
├── database.js         # Banco de dados centralizado do catálogo
└── style.css           # Estilização e animações premium
```
